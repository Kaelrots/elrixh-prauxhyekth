### 📝 카엘님의 최종 작업 루틴

<font style="font-weight:bold">1. 작업 준비 (기기를 바꿨을 때만)</font>

- 노트북 ↔ 데탑으로 넘어왔다면, 터미널에서 먼저 Web 폴더에서 <font style="font-weight:bold">`git pull`</font>을 한 번 쳐주세요. (웹 폴더의 깃허브 설정과 동기화를 맞추기 위함입니다.)
    ```
    git pull
    ```
- `Publish` 폴더는 구글 드라이브가 알아서 옮겨주니 초록색 체크(✅)가 떴는지 확인만 하시면 됩니다.

<font style="font-weight:bold">2. 글쓰기 및 빌드 (평소 작업)</font>

- 원본 폴더(옵시디언)에서 신나게 세계관 설정을 씁니다.
    
- `Publish` 폴더로 추출(Export)합니다.
    
- _(선택)_ 로컬(`localhost`)에서 먼저 확인하고 싶다면 터미널에 `npm run sync-build`를 돌려봅니다.

<font style="font-weight:bold">3. 깃허브 배포 (전 세계에 쏘기)</font>

- 터미널(Web 폴더)에서 아래 세 줄만 치면 끝납니다.
    
    Bash
    
```
  git add .
```
    
 ```
 git commit -m "오늘의 세계관 업데이트"
 ```
	 
```
git push origin main
```

# 🌌 엘루닉어(Ellunic) 폰트 제작 및 적용 가이드

이 문서는 Calligraphr와 Photoshop을 사용하여 커스텀 폰트를 제작하고, 이를 Obsidian Publish 환경에 웹 폰트로 적용하는 절차를 다룹니다.

## 1단계: 템플릿 준비 (Calligraphr)

가장 먼저 폰트 제작 사이트에서 '빈칸'이 뚫린 템플릿을 내려받아야 합니다.

1. <font style="font-weight:bold">웹사이트 접속:</font> [Calligraphr](https://www.calligraphr.com/ "null") 로그인
    
2. <font style="font-weight:bold">글자 구성:</font> `Template` 메뉴에서 <font style="font-weight:bold">불필요한 특수문자 삭제</font>.
    
    - <font style="font-weight:bold">권장 구성:</font> `A-Z` (특수형), `a-z` (기본형), `0-9` (필요시), ~~*기본 문장부호(`. , ! ?`)*~~[^1]
        
    - _Tip: 디자인하지 않을 특수문자는 과감히 삭제하세요. (삭제해야 시스템 기본 폰트로 출력되어 가독성이 유지됩니다.)_
        
3. <font style="font-weight:bold">다운로드 설정 (중요):</font>
    
    - <font style="font-weight:bold">File Format:</font> `PNG`
        
    - <font style="font-weight:bold">Draw helplines:</font> `Check (On)` (가이드라인 표시)
        
    - <font style="font-weight:bold">Characters as background:</font> `Uncheck (Off)` (배경 글자 끄기)
        

[^1]: 기본 문장부호를 따로 변형할 예정이 아니기 때문에, 기본 폰트가 출력되도록 설정 자체를 빼기로 함.

## 2단계: 디자인 및 그리기 (Photoshop) 🎨

다운로드한 PNG 파일을 포토샵에서 열고 작업합니다. <font style="font-weight:bold">이 단계가 퀄리티를 좌우합니다.</font>

### ⚠️ 절대 원칙 (Fail 방지)

- <font style="font-weight:bold">새 레이어 필수:</font> 원본(배경) 레이어에 직접 그리지 마세요. 반드시 <font style="font-weight:bold">새 투명 레이어</font>를 만들어 그 위에 그립니다.
    
- <font style="font-weight:bold">마커 보호:</font> 네 귀퉁이의 <font style="font-weight:bold">검은색 네모 마커</font>와 <font style="font-weight:bold">QR코드</font>를 절대 건드리지 마세요. (지워지거나 가려지면 인식 불가)
    

### 📐 가이드라인(4선지) 준수

칸 안에 보이는 희미한 가로줄 4개를 영어 공책처럼 활용하세요.

1. <font style="font-weight:bold">1번 줄 (천장):</font> 대문자(Shift 입력)나 위로 솟은 글자의 끝선.
    
2. <font style="font-weight:bold">2번 줄 (중간):</font> 소문자(기본형)의 머리 높이.
    
3. <font style="font-weight:bold">3번 줄 (⭐바닥/Baseline):</font> <font style="font-weight:bold">가장 중요.</font> 모든 글자의 발바닥을 이 선에 딱 맞춰주세요.
    
4. <font style="font-weight:bold">4번 줄 (지하):</font> 꼬리가 긴 글자(g, j 등)의 하한선.
    

### ⌨️ 키 매핑 전략 (User Custom)

- <font style="font-weight:bold">소문자 (a-z):</font> 엘루닉어의 <font style="font-weight:bold">기본 형태</font>를 그립니다. (평소 입력용)
    
- <font style="font-weight:bold">대문자 (A-Z):</font> 점이 찍히거나 장식이 추가된 <font style="font-weight:bold">특수 형태</font>를 그립니다. (Shift 키로 입력)
    
    - _Tip: 대소문자 구분이 없는 글자는, 대문자 그림을 복사해 소문자 칸에 똑같이 붙여넣으세요._
        

## 3단계: 폰트 생성 (Build)

1. <font style="font-weight:bold">사전 검수:</font> 작업한 이미지를 저장하기 전, <font style="font-weight:bold">가이드용으로 쓴 글자 레이어 등을 껐는지</font> 확인합니다.
    
2. <font style="font-weight:bold">업로드:</font> Calligraphr 사이트 > `My Fonts` > `Upload Template`
    
3. <font style="font-weight:bold">검토:</font> 업로드된 글자 미리보기에서 잘린 부분이 없는지 확인합니다.
    
4. <font style="font-weight:bold">생성 (Build Font):</font>
    
    - Font Name: `Ellunic` (공백 없는 영문 추천)
        
    - 형식: `.ttf` 또는 `.otf` 다운로드
        

## 4단계: 웹 폰트 변환 및 적용 (Obsidian Publish) 🌐

옵시디언 퍼블리시에서 방문자가 폰트를 볼 수 있게 하려면 웹 전용 포맷이 필요합니다.

### 1. 변환 (Convert)

다운로드한 `.ttf` 파일을 온라인 컨버터(예: CloudConvert)를 사용해 <font style="font-weight:bold">`.woff2`</font> 형식으로 변환합니다.

- _이유: woff2가 웹 로딩 속도가 가장 빠르고 호환성이 좋습니다._
    

### 2. 업로드 (Upload)

옵시디언 볼트 내 적절한 위치(예: `assets/fonts/`)에 `Ellunic.woff2` 파일을 넣고 <font style="font-weight:bold">Publish</font> 합니다.

### 3. CSS 적용 (`publish.css`)

`publish.css` 파일 최상단에 아래 코드를 추가합니다.

```
/* 1. 폰트 불러오기 */
@font-face {
    font-family: 'Ellunic';
    /* 경로 주의: 퍼블리시 루트 기준 경로 */
    src: url('lib/fonts/Ellunic.woff2') format('woff2'); 
    font-weight: normal;
    font-style: normal;
}

/* 2. 엘루닉어 전용 클래스 정의 */
.ellunic {
    font-family: 'Ellunic', sans-serif;
    font-size: 1.1em; /* 필요시 크기 조정 */
}
```

## 5단계: 사용 및 유지보수 🔄

### 사용법 (Markdown)

글 작성 시 HTML 태그를 사용하여 감싸주면 변환됩니다.

```
고대 기록: <span class="ellunic">hidden message</span>
```

- 평소엔 소문자로 입력 (`hidden`)
    
- 특수 강조가 필요할 때만 Shift 사용 (`HIdden`)
    

### 업데이트 전략 (무료 플랜 활용)

- <font style="font-weight:bold">수정:</font> 글자 모양이 마음에 안 들면 Calligraphr에서 해당 글자만 클릭(`Edit Character`)하여 이미지를 교체하세요.
    
- <font style="font-weight:bold">재배포:</font> 횟수 제한이 없으므로, 수정 후 다시 Build 하여 `woff2` 변환 -> 옵시디언 파일 덮어쓰기만 하면 즉시 반영됩니다.
    
- <font style="font-weight:bold">최종 진화:</font> 훗날 고퀄리티 폰트가 필요해지면, 전문가에게 의뢰하여 만든 파일로 `Ellunic.woff2` 파일만 교체하면 CSS 수정 없이 그대로 업그레이드됩니다.

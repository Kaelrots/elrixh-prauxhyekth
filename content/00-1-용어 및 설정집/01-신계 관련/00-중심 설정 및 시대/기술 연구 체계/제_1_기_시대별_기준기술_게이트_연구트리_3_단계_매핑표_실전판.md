# 제1기 시대별 기준기술(게이트) — 연구트리 3단계 매핑표 (실전판)

> 목적: 제1기 사건/사회 변곡점을 “연구트리 노드(게이트)”로 쓸 때, 시대별로 **최소 5～10개** 기준 기술/제도를 바로 꽂아넣을 수 있는 리스트.
>
> 작성 방식: ‘상징명’은 **문명(Civ) / 유로파(EU4)** 같은 시뮬레이션 게임에서 익숙한 명칭(또는 현실 역사 키워드)을 붙였고, 실제 구현은 엘리스 프라우셰크트 설정에 맞게 각색하면 됨.
> 
> **범위 안내:** 본 문서의 **제1기 시대별 기준기술(게이트)** 범위는 **01～09(후기 근대 시대)까지로 종료**하며, 이후 시대(근미래/미래/우주)는 **제2기 기술 문서에서 별도 정의**한다.

## 필드 설명
- **연구트리 매핑**: `코드 → 3단계 트랙(파일 원문 표기 그대로)`
- **TMC 추천**: `<ERA>-<TRL>`
  - ERA: `ANC / MED / EMO / IND / MOD / NFU / FUT / SPC`
  - TRL: 1～9 (연구트리 파일 정의 기준)
- **특수 오버레이(선택)**: `+ARC`
  - 의미: 차원/대전이/초상현상(3-8 등) 계열 “세계관 특수 게이트”
  - 표기 예: `MOD-7～9+ARC`
- **Ref 키워드**: 이후 노드 설명/본문 작성 시, 참고할 외부 레퍼런스 단서

# 0. 개요
- **목적**: 전시·사변 및 이에 준하는 **국가비상사태**에서 연방 최고평의회·연방 황실이 행사하는 **긴급조치권**(긴급황제령·비상명령)의 **트리거→의결/발령→공포·집행→사후통제** 전 과정을 표준화.
- **헌법 근거(발췌)**: 권리 제한과 비상사태 기본 원칙(총강 §15)과 전쟁·국가안보 장(제11장) 연계. 긴급명령·황제령은 **관보·전자고시 의무** 및 **사후 추인**·**일몰(sunset)**·**사법 우선심리** 원칙에 따름.
- **상위 파이프라인 연동**: 행정 파이프라인 v0.1(의결→황제령/행정명령→공포/집행→사후통제), 입법정비 신속 트랙(30일 자동상정), 헌법/일반 재판의 **우선심리** 라우팅에 연결.

<hr class="hr-thick-4">

# 1. 정의·등급(비상 사태 분류)
- **전시(WAR)**: 합법적 전쟁선포 또는 무력공격을 받은 상태. (제11장 링크)
- **사변(HOSTILITY)**: 전면전 직전/국경충돌·우주항로 교전·대규모 테러·사이버·마법공격 등 국가안보 중대 침해.
- **특별재난(EMER)**: 대행성충돌·대규모 역병·행정체계 마비 등 비군사 비상.
- **등급**: `α(전면) / β(광역) / γ(국소)` — 등급별 정족·권한·기간·통신제한 수위 차등.

<hr class="hr-thick-4">

# 2. 트리거(개시 사유) 및 워치 신호
- **개시 트리거**: (a) 합법 전쟁선포 절차 개시·의결, (b) 적의 선제공격 탐지, (c) 헌법상 공포의 위험·국가 존립 위협, (d) 특별재난 선포 필요.
- **워치 신호(T0)**: 상원 헌법위원회·총의회 의사국·디지털의정청·선관위·감사원·대법원·헌법원·정보국에 **일괄 워치** 발신 → **우선심리/강제트랙** 및 **전자관보 예비 슬롯** 자동 확보.

<hr class="hr-thick-4">

# 3. 합동 상황심사(최대 24h/긴급 6h)
- **구성**: 최고평의회 비상상황실(의장단) + 국방·외교·정보·재정 + 법무(헌법·권리 영향) + 전산/통신·보건 등 사건 유형별 TF.
- **산출**: ① **법률·헌법 적합성 검토서**, ② **작전/행정 실행계획(ROE·SOP)**, ③ **권리제한 영향표(RIA-EM)**, ④ **재정영향(FIA-EM)**, ⑤ **전자고시 번들**(관보용 전문·요약·FAQ·API 스키마).

<hr class="hr-thick-4">

# 4. 최고평의회 비상 의결(전자긴급회의 허용)
- **정족수**: 통상 과반(행정 파이프라인 기준)이나, **α/β 등급 긴급**은 **출석 1/2 이상 + 출석 2/3 찬성**으로 **임시의결** 가능. 임시의결은 **≤7일** 내 본정족 재심의.
- **의결항목**:
  1) **긴급황제령** 발령 요청(권리 제한·동원·배급·검열·검문·봉쇄 등 포함 범위·기간·지역 명시)
  2) **비상행정명령** 패키지 승인(부처별 집행명령·예산전용 한도)
  3) **전쟁선포/동원** 안건(제11장 라우팅 병행)
  4) **전자관보·API 고시** 승인(즉시 공포·시행 시각 명시)
  5) **사후 추인·일몰** 타임라인 확정

<hr class="hr-thick-4">

# 5. 황실(황제) 발령 — 긴급황제령·합동선언
- **발령권**: 최고평의회 의결에 따라 **황제의 신속 발령**. 긴급황제령은 **관보·전자관보 동시 공포**로 즉시 효력.
- **형식**: 조문-별표-부칙 + **권리제한 체크리스트** + **선언문**(합동선포 의전 규정 준수) + **머신판**(API/e-Form 스키마 ID 포함).
- **범위·한계**: ① 권리장전 **본질 침해 금지**, ② **등급별 최대기간**(α=≤30일, β=≤21일, γ=≤14일), ③ **지역·업무 한정성**, ④ **비례·필요·최소침해** 원칙.

<hr class="hr-thick-4">

# 6. 공포·전자고시·정보관리(≤1h 내)
- **관보/전자관보**: 전문·요지·지침·FAQ·지도/구역 JSON·일몰일자 표시.
- **API 고시**: `EMER-<연도>-<연번>` / `WAR-<연도>-<연번>` 규칙, 호출 제한·보안등급 표기(Public/Gov-Internal/Secret).
- **통신/언론**: 검열·방송명령은 **판사후 통제** 원칙(사후 72h 내 통지·기록, 우선심리).
- **기밀관리**: §16 기준 등급 부여, 상·하 공유 범위 구분, 황제 직접 재가 요건 명시.

<hr class="hr-thick-4">

# 7. 집행·동기화
- **집행계획 롤아웃**: 부처·군·지방에 ROE/SOP·권한위임 고시, **예산 전용 한도** 공시.
- **정책 동기화**: 선거·정당·윤리·부패·국가안보 사건은 **국가·전국(3-3) 강제 트랙** 라벨링 → **고등(1심전담)→중앙→대법** 경로.
- **데이터**: 상황판 API(집행지시/성과/KPI), 감사로그 의무.

<hr class="hr-thick-4">

# 8. 사후통제(다중 가드레일)
- **의회 추인**: 긴급황제령 발령 **≤7일** 내 **상·하원 동시 보고**, **≤30일** 내 **총의회 추인 표결**. 부결 시 **소급 배제(신뢰보호 제외)**·즉시 종료.
- **상원 헌법위원회**: 상시 **철회/수정 권고** 가능, 최고평의회 **재심 의무**.
- **사법 우선심리**: 헌법원·대법 **우선 배당**·신속 선고. 권리제한 집행정지 요건·범위 표준화.
- **감사원·정보공개**: 집행 **주간보고** 공개(전략·기밀 제외), **사후 백서** 1년 내 제출.
- **일몰·연장**: 일몰 도래 72h 전 자동 경보, 연장은 **동일 정족의 재의결 + 황제 재발령** 필수.

<hr class="hr-thick-4">

# 9. 전쟁선포·동원 연계(제11장 교차)
- **전쟁선포 플로우(상세)**
  1) **개시안**: 최고평의회 결의(사유·목표·전장·기간·ROE 초안 첨부)
  2) **상원 헌법위원회 예비심**: 합헌성·권리영향·대외법 정합(과반 의결로 본회의 회부 권고)
  3) **총의회 본회의**: *2/3 출석·2/3 동의*로 전쟁선포 의결 → 황제 **합동선포** 준비
  4) **황실 의전·합동선포**: 황제 선포 + 최고평의회 공동선언(관보/전자관보 동시 공포)
  5) **사후 동기화**: 비상경제·동원·징발·검열 등 하위 **긴급황제령/비상행정명령** 즉시 발령·연동

- **동원·징발 플로우(상세)**
  1) **범위 설계**: 인적(예비군·민간전력), 물적(장비·우주선·에너지·식량) 카테고리·지역화
  2) **권리·보상 설계**: 보상산식·우선임대·대체 의무·면제 사유·취약계층 보호규칙
  3) **집행 명령**: 부처별 집행SOP·ROE, 지방정부 역할·타임라인, 분쟁조정 창구
  4) **쟁송·구제**: 즉시항고·집행정지 기준, 우선심리 라우팅(헌법원/대법)

- **보상·쟁송 API(머신판) — 예시 스키마**
  - `POST /api/war/declare` — { id, title, legal_basis, theaters[], roe_id, start_at, sunset_at }
  - `POST /api/mobilize` — { mobilization_id, scope: {human[], material[]}, region[], start_at, legal_basis, compensation_formula_id }
  - `POST /api/requisition` — { requisition_id, asset_type, owner_id, region, duration, compensation_estimate }
  - `POST /api/compensation/claim` — { claim_id, person/org, basis, amount_requested, docs[], bank_info }
  - `POST /api/appeals` — { appeal_id, decision_id, grounds, relief_sought, urgency }
  - 공통 메타: { security_level, version, hash, signature, notified_entities[] }

- **필수 기재항목(긴급황제령 본문)**: 전장·지역·기간(일몰), 권리제한 항목과 한계, 보상·구제 창구, 데이터 공개 범위, 신민 보호조치.

# 10. 디지털·양식 표준(머신판)
- **식별자**: `DEC-EMER-α/β/γ-YYYY-####`, `ORDER-EM-<부처>-vX.Y`.
- **서식 번들**: (A) 전문, (B) 집행지침, (C) 권리제한 체크리스트, (D) RIA/FIA, (E) 지역·대상 JSON, (F) FAQ, (G) 일몰 타이머 메타.
- **샌드박스**: 모의 호출용 엔드포인트·레퍼런스 구현 공개.


## 권리제한 항목 표준(체크리스트)
> 모든 항목은 **필요·비례·최소침해** 원칙을 개별적으로 충족해야 하며, **RIA-EM**(권리영향분석)과 **판사후 통제** 기준을 함께 기재한다.

- **통행·이동**  
  - [ ] 통행금지/통제 구역·시간대  
    - **기준**: α=전면 야간(예: 22:00~05:00) 또는 전구역, β=핵심구역·관문, γ=국소 위험지대.  
    - **범위**: 행정구역 코드+작전구역 코드 병기(지도 JSON 포함).  
    - **예외**: 의료·구조·필수근로·대피 이동은 허가증·QR 패스 발급.  
    - **구제**: 현장 이의제기→즉시기록→24h 내 행정심사, 위법 제재 취소·보상 절차 고지.  
  - [ ] 검문소 설치·우회 동선  
    - **배치**: 주요 관문·교차로·항행 포인트, **대기시간 KPI** α≤15m / β≤10m / γ≤7m.  
    - **데이터**: 통과량·차단율·평균대기 산출, 취약계층 우선통로 표기.  
  - [ ] 대피/피난 명령(취약계층·장애인 지원 계획)  
    - **발령**: 위험등급·풍향/전선 예측 기반 단계별(권고→의무).  
    - **지원**: 수송수단·보조기기·동행 지원 인력, 임시주거·식량 배급 연동.  

- **집회·결사·표현**  
  - [ ] 집회 제한 요건·구역·시간  
    - **요건**: 폭력·파괴 고위험·작전 방해·감염확산 우려. **대체수단**(소규모·분산·온라인) 제시 의무.  
    - **구역**: 군·관공서 100m 안전지대(예시), 완충지대 지도 JSON.  
  - [ ] 선전·선동물 제한 범위(명확성·과잉금지 검토)  
    - **범위**: 군사기밀, 폭력 직접·현존 위험 유발 표현. 일반적 정치·비판표현은 **허용** 원칙.  
    - **절차**: 차단·삭제 시 **사후 통지 72h**·열람권·불복 절차 API 제공.  

- **검열·통신·언론**  
  - [ ] 네트워크 차단·속도제어·특정 플랫폼 제한  
    - **수단**: 도메인·프로토콜 단위 제한(화이트/블랙리스트), 응급·공공서비스 **우선 보장**.  
    - **기한**: α≤48h, β≤24h, γ≤12h(연장 시 사유 고지).  
  - [ ] 방송명령·정정보도·공익방송 시간 배정  
    - **명령**: 재난경보·대피안내 우선 송출, 공익 슬롯 배정 기준 공개.  
    - **정정보도**: 허위정보 판정 근거·증빙 공개, 불복 채널 명시.  
  - [ ] **사후 통지/기록 72h 내** 원칙  
    - **기록**: 차단·검열·명령 로그 해시·타임스탬프 관보 교차보관, 열람 등급 Public/Gov-Internal/Secret.  

- **검문·수색·체포**  
  - [ ] 영장 요건/예외, 신체·소지품·거주 수색 범위  
    - **예외**: 현행범·급박위험·작전구역 출입 시 제한적 예외(범위·시간 엄격 한정).  
    - **증빙**: 바디캠/로그 의무, 부적법 수색 증거배제 원칙 유지.  
  - [ ] 임시구금 기간·접견권·변호권 보장  
    - **상한**: α≤48h, β≤36h, γ≤24h(판사 사후심사).  
    - **권리**: 통역·의료·변호 접견권, 취약계층 보호조치 기록.  

- **데이터·감시**  
  - [ ] 위치·트래픽·금융데이터 수집 범위·보존기간  
    - **최소수집**: 목적·필드 최소화, α/β/γ별 보존 31/21/14일(기소 시 별도).  
    - **비식별**: 통계 공개 시 가명처리·k-익명성 기준.  
  - [ ] 자동판단·프로파일링 사용 제한·감사로그  
    - **금지**: 순수 자동처분 금지(최소 1회 인간심사), 고위험 모델은 모델카드·바이어스 리포트 공개.  
    - **감사**: 호출·결정 로그 365일+, 무결성 해시 관보 교차검증.  

- **경제·배급·가격**  
  - [ ] 배급 품목·우선순위·카드/토큰 발급  
    - **우선**: 의료·구호·영아·고령·장애·임산부 > 일반 신민. 디지털 토큰/카드 발급·분실 재발급 SLA 24h.  
  - [ ] 가격표시·상한·조달특례·사유재산 영향  
    - **상한**: α 단기 상한·비축 방출, 사유재산 수용 시 **정당보상** 산식·불복 절차 병기.  
    - **조달**: 긴급조달·입찰특례·부패 방지 로깅.  

- **동원·징발**  
  - [ ] 인력 소집 등급·면제 사유·대체복무  
    - **등급**: M1(전면)·M2(부분)·M3(특정기술). 학업/장애/필수돌봄 **면제·유예** 기준 명시.  
    - **권리**: 소집통지 열람·이의·대체복무 신청 API, 기한 α≤72h/β≤96h/γ≤120h 처리.  
  - [ ] 물자·시설·우주선 징발 범위·기간·보상산식  
    - **범위**: 정격출력·톤수·용도별 표, **기간** 상한·반납 조건.  
    - **보상**: 기준가+감가+운휴+위험가중 산식 공개, 분쟁조정·재판 우선심리.  

- **보건·검역**  
  - [ ] 출입국·행성 간 이동 제한  
    - **코리더**: 필수 물류·의료 코리더 유지, 음성/면역 증명 요건.  
  - [ ] 검역·격리 기준·의료자원 배분 규칙  
    - **격리**: 접촉도/증상에 따른 3단계, 재검 주기·해제 기준 명시.  
    - **배분**: 중환·응급 우선, 윤리위원회·로그 공개(개인정보 보호).  

- **기타**  
  - [ ] 교육·사법·행정 서비스 축소·연기 기준  
    - **축소**: 원격전환 우선, 필수 시험·기한 연장 규칙. 사법 절차는 **우선심리** 대비 비대면 병행.

# 1. 재판의 유형(본류→전문)

## 1-1. 본류(Top-Level)
- **민사**: 채권·물권·불법행위·회사/상사·지재(민사)·가사/상속 등
- **형사**: 경범~중범·경제/조직범·국가안보범 등
- **행정**: 취소·무효·부작위·의무이행·조세·출입국·공공요금·규제 사건 등
- **헌법(집중형)**: 위헌법률심판·헌법소원·권한쟁의·탄핵(연방 헌법원 전속)

## 1-2. 전문/특수 영역(본류 하 상설 전문부로 처리)
지식재산(특허/상표/저작), 노동, 가사/소년, 파산/회생, 조세, 환경·집단소송, 해상·국제거래·중재, 선거·정당, 데이터·플랫폼 규제, 공직자윤리·징계(준사법) 등

> 관점: 1-1의 **본류** 위에 1-2의 **전문부 태그**를 붙여 라우팅/배당.

# 2. 재판 유형별 담당 법원 분과(급별 표준 조직)

## 2-1. 연방직할구·연방자치구 **기초지방법원**(1심 단독 중심)
- **민사단독부**(소액·단독·집행/보전)
- **형사단독부**(경미/약식/즉결)
- **비송부**(등기·후견 등)
- **조정·중재지원실**(조정전치·내부중재 연계)
> 옵션 간이전문부: 가사, 근로(임금), 소액지재

## 2-2. 연방직할조계·연방자치조계 **중등지방법원**(합의 1심 허브)
- **민사합의부 / 형사합의부 / 행정부**
- **전문부(상설)**: 지재·노동·가사/소년·파산/회생·조세·환경/집단·해상/국제·선거
- **보전·집행 전담부**(광역 보전·강제집행)
- **사건관리국**(기록·기일·전자송달)

## 2-3. 연방직할주·연방자치주 **고등지방법원**(2심 항소 전담)
- **민사/형사/행정 항소부**, **전문항소부**(지재·조세 등)
- **집중조정부**(상소심 조정·화해)
- **선도법리검토실**(하급심 불일치 모니터)

## 2-4. **연방 중앙지방법원**(특별 상소심·재항소심 / 전국 통일 판단 허브)
- **민사·형사·행정 재항소부**(고등 판결 재항소)
- **전국집중부**: 조세/공공요금·데이터/플랫폼·대규모 집단/환경·선거/정당·국가안보·연방기관 사건
- **전원합의부(중앙)**: 법리통일 필요사건 지정 심리
- **헌법제청실**: 위헌 의심 규범 즉시 제청(헌법원 연동)

## 2-5. **연방 대법원**(최종심·법리 통일)
- **상고부**(민/형/행), **전원합의체**(법리 변경·통일)
- **상고허가국**(허가 심사: 중요 법리·판례불일치·중대한 절차위반)

## 2-6. **연방 헌법원**(집중형 위헌심사)
- **헌법재판국 / 헌법준수심의국 / 헌법사무국**

# 3. 재판 유형별 사건 유형 기준(배당 트랙)
> 원칙: **가장 높은 등급 기준을 우선 적용(상향배당)**

## 3-1. **경미(경)**
- 민사: 통일 기준금액 **X 이하**, 다툼 단순, 증거 간명
- 형사: 경범·약식·즉결 대상
- 행정: 소액 과태료·경미 행정질서위반(파급 미미)
- **배당**: 기초(1) → 고등(2) → **대법(3)**

## 3-2. **중대(중)**
- 민사: X 초과, 다수 당사자/고난도 증거, 사회적 파급(권역 이상)
- 형사: 중범·경제/조직범·대형 화이트칼라
- 행정: 고액 조세·대형 인허가·규제 영향 광범위
- **배당**: **중등(1)** → 고등(2) → **대법(3)**

## 3-3. **국가·전국(집중관할)**
- 당사자: 연방기관·준/독립기관·국가급 공기업·전국 사업자
- 쟁점: 조세·관세·예산/기금·전국 공공요금·데이터/플랫폼·선거/정당·국가안보·대규모 집단/환경
- 파급: 전국 동일 법률문제, 복수 권역 동시 영향
- **배당**: **고등 1심전담부(1)** → **중앙(2)** → **대법(3)**

## 3-4. **헌법쟁점 포함 일반사건(사이드 제청)**
- 제청 3요건: **직접성**(사건 전제) / **현재성**(현효) / **중대성**(결정적 영향)
- 흐름: (경·중·국가 트랙 진행 중) 해당 심급이 **헌법원 제청** → 헌법원 **O/X** → 본안 재개 → 최종 **대법**
- 효력: 위헌 확정 시 해당 규범 적용 배제 + 입법·행정 정비(SLA) 자동 통지

# 4. 사건 유형별 재판 진행 절차(표준 플로우)

## 4-1. **민사**(경/중/국가 트랙 공통)
1) 소 제기(전자) → 2) 송달/답변 → 3) 변론준비(쌍방 서면교환) → 4) 본안 심리(구두+서면, 증거조사)
→ 5) 1심 판결(기초/중등/고등1심전담) → 6) 항소(고등) → 7) 상고허가 신청 → 8) **대법 선고** → 9) 확정·집행(보전·강제)

## 4-2. **형사**
0) 수사(경찰/검찰)·공소 제기 → 1) 공판준비(구속 여부 포함)
→ 2) 공판 심리·증거조사·(필요 시 배심 보조) → 3) 1심 판결(경=기초 / 중=중등 / 국가=고등1심전담)
→ 4) 항소(고등) → 5) 상고허가 → 6) **대법 선고** → 7) 확정·형 집행 / (재심 요건 충족 시 별도 절차)

## 4-3. **행정**
1) 처분 통지 → (선택) 전치/이의 → 2) 소 제기(행정부) → 3) 심리·증거조사
→ 4) 1심 판결(중등 또는 고등1심전담/중앙집중사안은 ‘국가·전국’ 경로) → 5) 항소(고등) → 6) 상고허가 → 7) **대법 선고**

## 4-4. **헌법(집중형)**
- **순수 헌법사건**(헌법소원·권한쟁의·탄핵·추상 위헌심사): **헌법원 단독**(대법 비개입)
- **제청 사건**: 본안 재판부가 헌법원에 제청(전자연동) → O/X 확정 → 본안 이어서 선고(최종 **대법**)

> 전자법원 연동: 제청 시 전자기록 원클릭 송부(사법행정처↔헌법사무국), 공개기록청은 O/X 결과를 본안 사건 페이지에 자동 표기.

# 5. 재판의 기본 원칙(운영 규범)
1) **성문법 우위와 예측가능성**: 헌법 최상위, 법률·황제령·총의회/최평 결의·최종판결(2층) → 행정명령 → 지방규범.
2) **최종심 대법원 일원화**: 모든 일반사건의 최종심은 **연방 대법원**.
3) **집중관할·상향배당**: 국가·전국 사건은 고등1심전담→중앙 집중, 경계값 초과시 상향배당.
4) **판례 통일장치**: 중앙 ‘선도판결’ + 대법 ‘전원합의체’로 법리 통일, 법률해석국·양형위와 즉시 연동.
5) **공개성·익명화·데이터화**: 선고문 공개·익명화·메타태그(본류/전문/트랙/제청) 부여.
6) **적정절차·방어권 보장**: 무죄추정·과잉금지·비례·변호권·증거개시(형사)·당사자평등.
7) **전문화와 신속**: 전문부 상설·IT 심리(원격·전자송달)·SLA 기반 기일관리.
8) **헌법합치 원칙**: 2층 내부 충돌 시 **최종판결·헌법 판단 합치**로 정비.

# 6. 재판 유형별 세부 진행 절차(체크리스트)

## 6-1. 민사 세부
- **입구**: 소가·난이도·파급 지표 산정 → 경/중/국가 트랙 자동 추천
- **전형 쟁점관리**: 문서열람권·증거개시(쌍방)·증거보전·전자증거 포맷 규칙
- **보전/집행**: 가압류·가처분 표준 양식 / 강제집행 포털 연동
- **집단/대표소송**: 인증요건·통지·합의 승인 절차
- **상고 포인트**: 중요 법리·불일치 선별, 전합 필요성 건의 라인

## 6-2. 형사 세부
- **수사·기소**: 영장주의·변호인 접견·자백보강·위법수집 증거배제
- **구속·보석**: 비례·필요최소, 보석 사유·조건 표준화
- **공판**: 증거개시, 증인 신빙성 평가, 배심 보조(선정/설명 템플릿)
- **양형**: 양형기준표·가이드라인(양형위 연동), 피해자 의견청취·배상명령
- **재심/보상**: 재심 사유·절차, 형사보상·명예회복

## 6-3. 행정 세부
- **전치주의**: 이의·심판 절차 가이드(생략/필수 구분)
- **집행정지**: 본안과 병행 심문, 회복곤란 요건·공공복리 비교형량 시트
- **조세**: 과세표준·법령해석 일관성(법률해석국 코멘트 즉시연동)
- **규제·공공요금**: 전국 동형 사건은 중앙 집중관할 적용 체크

## 6-4. 헌법 세부
- **제청 문턱**: 직접성·현재성·중대성 점검 체크박스
- **심리**: 서면 중심+공개변론(필요시), O/X 선고만
- **후속**: 위헌 확정 → 입법·행정 정비기한(SLA) 자동 통보, 경과규정 안내

## 6-5. 전문/특수 공통
- **지재**: 기술설명서·클레임차트 표준, 감정인 풀
- **노동**: 집단분쟁 조정·긴급가처분 트랙
- **가사/소년**: 비공개 심리·후견인/조사관 보고 템플릿
- **파산/회생**: 채권자집회·인가요건 체크리스트
- **해상/국제**: 준거법·관할합의·중재 병행 심리 가이드
- **선거/정당**: 단기 심리·확정 선고 시한
- **데이터/플랫폼**: 포렌식 표준·알고리즘 증거제출 지침

# 부록 B. 입법·위임 체계(헌법/연방 법률/연방 황제령)

> 목적: 본 문서의 재판 유형·배당·절차가 안정적으로 작동하도록 **상위 규범(헌법)**, **실체·절차 법률(연방 법률)**, **시행·운영 규정(연방 황제령)**의 계층별 근거를 미리 설계한다.

## B-1. **연방 헌법**에 둘 조항(제목·핵심 내용 / 본문 수록 위치)
- **제○조(사법권의 귀속과 범위)** — 사법권 연방 귀속, 일반사건 최종심=연방 대법원.  
  **수록 위치:** 제6장(사법 기관) 제1절 총칙, *제6장 제1조*.
- **제○조(헌법재판의 집중)** — 위헌심사·헌법소원·권한쟁의·탄핵은 **연방 헌법원 전속**.  
  **수록 위치:** 제6장 제2절 헌법재판, *제6장 제2조*.
- **제○조(법원 체계와 심급)** — 기초·중등·고등·중앙·대법 심급, **중앙=특별 상소심(재항소)**.  
  **수록 위치:** 제6장 제1절 총칙, *제6장 제3조*.
- **제○조(상고허가와 전원합의체)** — 상고허가 사유(중요 법리·판례불일치·중대한 절차위반), 전원합의체 법리변경.  
  **수록 위치:** 제6장 제3절 대법원, *제6장 제7조*.
- **제○조(헌법 제청)** — 제청 3요건(직접성·현재성·중대성), 헌법원은 **O/X**만 판시.  
  **수록 위치:** 제6장 제2절 헌법재판, *제6장 제4조*.
- **제○조(위헌확정의 효력과 후속의무)** — 위헌 규범 적용 배제 + **정비 기한(SLA)**로 의회·행정부·황실 후속의무.  
  **수록 위치:** 제6장 제2절 말미, *제6장 제5조* (※ 제5장(입법)·제4장(행정) 관련 조항에 **연동 규정** 교차배치).
- **제○조(공개와 적정절차)** — 재판 공개·익명화·변호권·무죄추정·비례·전자소송 기본원칙.  
  **수록 위치:** 제1장(권리장전) 절차보장 파트, *제1장 제12조* 신설.
- **제○조(전문화와 전자사법)** — 전문부 설치·전자기록·원격심리의 헌법 근거.  
  **수록 위치:** 제6장 제1절 총칙, *제6장 제6조*.
- **제○조(규범 위계)** — 헌법> (법률=황제령=최종판결=의회결의) > 행정명령 > 지방.  
  **수록 위치:** *총강 제8조* (현행 유지, 필요 시 “최종판결” 문구 명시 강화).
- **제○조(선거·투표·정당·공직윤리·부패 사건의 집중관할)** — 해당 사건은 **고등(1심전담) → 중앙(재항소) → 대법(최종)** 강제 트랙.  
  **수록 위치:** 제6장 제1절 총칙, *제6장 제8조*; 제13장(선거·정당) 말미에 **교차 규정** 신설.

## B-2. **연방 법률**(패키지) — 제정/개정 권고 조항
1) **연방 사법조직 법** — 법원 체계·관할·분과, **중앙 재항소심**·**전국집중부**·**고등 1심전담부** 근거, 전문부 상설.
2) **민사/형사/행정 소송 법** — **배당 트랙 규정**(경=기초, 중=중등, **국가·전국=고등→중앙→대법**), 상향배당·집중관할 리스트 법률화, 전자절차·증거개시·보전/집행 통합.
3) **헌법재판 절차 법** — 제청 3요건, 접수·평의·선고 기한, O/X 한정 판시, 사건연동 API.
4) **전자법원 운영 법** — 전자소송·송달·기록·원격심리·증거 표준·보안.
5) **판례공표 및 법률해석 연계 법** — 선도판결 지정, 판례번호·메타태그, 익명화 기준, **법률해석국/양형위** 연동.
6) **집단·대표소송 법** — 인증요건·통지·합의 승인·금지명령 병행.
7) **증거 법(디지털증거 포함)** — 포렌식·알고리즘 제출·비밀유지명령·위법수집 배제.
8) **선거·투표·정당·공직자윤리·부패 특례 법** — **강제 트랙**·단기 심리·확정 선고 시한·증거개시 특례·공익소송.
9) **양형위원회 법** — 구성·의결·기준 제정·갱신·판결 자동연계.
10) **공개기록·익명화 기준 법** — 공개 범위·비실명화·열람 API·연구용 2차 이용.

## B-3. **연방 황제령**(시행령·규칙·고시) — 위임 사항(예시)
- **사건 배당·상향배당 규칙령** — 경/중/국가·헌법쟁점 산정표, 파급지수 가중치, 자동배당 파라미터.
- **집중관할 부속표 A 고시** — 조세·관세·전국 공공요금, 데이터/플랫폼, **선거·투표·정당·공직자윤리·부패**, 대규모 집단·환경, 연방기관·준독립기관 피고 사건.
- **상고허가 심사 지침** — 중요 법리·불일치·절차위반 체크리스트, 거절사유 표준문안.
- **선도판결 지정·전원합의부 회부 절차규칙** — 지정요건, 의견서 제출, 공개변론 운영.
- **헌법 제청 표준서식·기한 고시** — 제청서 구조, 전자연동 메타데이터, 병합/분리 기준.
- **전자기록 기술규격 고시** — 파일 포맷, 메타태그(본류/전문/트랙/제청), 서명·타임스탬프.
- **판례·해석·양형 동기화 SLA** — 대법/중앙 선고 → 해석국·양형위 고시까지 최대 기한.
- **공개기록·익명화 세부기준** — 비실명화 정책, 제3자 열람·데이터셋 배포 규칙.
- **선거·투표·정당·공직자윤리·부패 강제 트랙 시행령** — 접수 즉시 트랙 라벨링, 타 트랙 지정 금지 예외 사유.

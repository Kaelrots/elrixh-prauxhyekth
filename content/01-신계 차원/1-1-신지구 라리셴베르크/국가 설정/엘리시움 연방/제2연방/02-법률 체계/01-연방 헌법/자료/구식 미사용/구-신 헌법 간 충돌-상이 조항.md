# 1) **실질 충돌(내용 충돌/우선 규범 선택 필요)**

- **황권 승계 절차** — 신헌법 제21조 / 기존 제1장 제3조  
    **신헌법**: 승계위원회 지명·선언 → 황실 재가 → 헌법원 직무적부심사 확정. 절차 불능 시 총의회 선출(결선 반복), 피선거권은 직전 3대 이내 직계 한정. 섭정 규정 포함.  
    **기존**: 재임 중인 황제가 **제1황위계승자 ‘지명’**으로 승계 확정.  
    → **판정: 충돌**(지정 주체·확정 절차가 다름). 신헌법 제21조 체계를 본문 제1장에 반영하거나, 기존 ‘황제 단독 지명’ 규정을 폐기/보완하는 정합화 필요.
    
- **개헌 절차의 전면 불일치** — 신헌법 제20조 / 기존 제5장 제10·11·30조  
    **신헌법**: 헌법원 개정권고 → 상원 헌법위 회신·정리 → 최고평의회 특별심의 → 황실 헌법자문위 최종 확정 → 상원 헌법위 총의회 회부·공표 → 황실·황제 재가 즉시 효력.  
    **기존**: 상원 헌법위원회 발의 → **헌법제정소**(제정위원 50명 전원 출석·과반 동의) 최종 반영·의결 → 황제 승인(또는 앞·뒤 단계에서 역할 상이).  
    → **판정: 충돌**(기관 구성·의결 정족·선후 절차 상이). 신헌법 제20조 절차로 일원화하거나, 헌법제정소의 지위를 총강 제20조 체계 속에 재배치해야 함.

# 2) **내용 상이(병존 가능하나 운영·위계 정리 필요)**

- **정당 해산·소멸 요건의 이원화** — 신헌법 제10조 / 기존 제2장 제7조  
    **신헌법**: 위헌 정당 해산은 **헌법원 건의 + 황제 재가**.  
    **기존**: **원내 의원 ‘0명’ 자동 해산** 규정.  
    → **판정: 상이**(사유·주체·절차가 다름). 병존 가능하나, ‘자동 해산’이 신헌법의 해산 전속절차(헌법원+황제)와 충돌하지 않도록 **조문 연결·우선순위** 명시 필요.
    
- **준비 기금의 독립성 vs 명령 위상** — 신헌법 제12조·제8조 / 기존 제4장 제7·9조  
    **신헌법**: 준비 기금은 독립기관, **법규 위계**는 헌법>법률=황제령>…  
    **기존**: **기금위원회장 명령 = 황제령**(동위).  
    → **판정: 상이**. ‘동위’ 자체는 제8조 체계와 합치하나, **독립기관의 명령 발동 범위**와 통제(감사·사후심사)를 총강/경제장에 교차 명문화 권고.
    
- **전쟁·파병 절차의 기술 층위 차이** — 신헌법 제7조 / 기존 제3장 제10조·제2장 제2·4조  
    **신헌법**: 자위권 범위 내 사용 + **상호 견제 의무**의 원칙 선언.  
    **기존**: 구체적 **의결 루트**(최고평의회→상원 헌법위 과반→총의회 2/3).  
    → **판정: 상이(보완관계)**. 원칙(총강)과 절차(본문)의 연결 조문을 넣어 **중복/모순 방지** 필요.
    
- **국교·신앙 자유 조항의 중복** — 신헌법 제5조 / 기존 제2장 제1조·제8장 제15조  
    → **판정: 상이(중복)**. 내용은 합치. 총강에 원칙만 두고, 구체 금지(의회 권한 제한)는 입법 장에 두도록 **위치 정리** 권고.
    
- **법규 위계·동위 규정의 반복** — 신헌법 제8조 / 기존 제3장 제11·12조  
    → **판정: 상이(중복)**. 본질 합치. 다만 기존 제12조의 **행정명령 무효 통로**(상원 헌법위·하원 사법위·대법원)와 총강 제8조의 **동위/우선 원칙** 간의 **참조 규정**을 서로 삽입 권고.
    
- **권리장전 우선·비상사태·기밀 운영 세부** — 신헌법 제14·15·16조 / 기존 제8장 제1·16·20~22·29~31조  
    → **판정: 상이(세부 강화)**. 신헌법은 **일몰·통지·기록** 등 절차를 명시. 본문 권리장전 조항에 **절차적 통제·사후 심사**로 맞춰주는 편이 정합성 높음.

# 3) **조치 제안(에디트 플랜 초안)**

1. 본문 **제1장(황실·승계)**에서 기존 제3조를 신헌법 제21조에 맞게 **개정**: ‘황제 단독 지명’은 ‘승계위원회 지명 + 헌법원 직무적부 심사 + 황실 재가’로 전환, 총의회 백업선거·섭정 규정 신설.
    
2. 본문 **사법/헌법 파트**에서 기존의 **헌법제정소(제정위원 50명)** 지위를 재정의: 총강 제20조의 다기관 절차 속 ‘특별 심의·의결부’로 흡수 또는 별도 부칙으로 **폐지·경과 규정** 삽입.
    
3. **정당 조항**: ‘자동 해산(의원 0명)’을 **당내 법적 소멸**로, ‘위헌 해산’을 **국가적 해산**으로 구분하는 정의 조항 추가.
    
4. **준비 기금 명령 위상**: ‘황제령 동위’ 유지하되 발동 범위·감사 루프(상원 세출·하원 재정·대법원 사후심사)를 본문 경제 장에 보강, 총강 제8조에 **연동 문구** 삽입.

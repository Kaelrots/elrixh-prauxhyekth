# 1. 헌법재판의 유형(본류 → 세부)

## 1-1. 본류(Top-Level)
- **위헌법률심판**: 법률·명령·결의 등 상위규범의 **헌법합치성** 판단(O/X).
- **헌법소원**: 공권력의 작용 또는 부작용으로 인한 **기본권 침해** 구제.
- **권한쟁의심판**: 국가기관·지방·독립기관 상호 간 **권한의 존부·범위** 확정.
- **탄핵심판**: 탄핵소추를 받은 공직자의 **파면 여부** 결정.

## 1-2. 세부 유형(접수 경로 기준)
- **법원 제청형**(위헌법률심판): 일반사건 재판부가 위헌 의심 규범을 **선결문제로 제청**.
- **직접제소형**(위헌법률심판·헌법소원·권한쟁의·탄핵): 당사자·기관이 요건 충족 시 **직접 신청**.
- **추상/구체 구분**: 구체적 사건 전제 여부에 따라 **구체적 규범통제/추상적 규범통제**로 구획.

> 판시 범위: 헌법원은 원칙적으로 **O/X(위헌/합헌) 판단과 권한확정/파면 여부**만 판시하며, 사실인정이나 개별 사건의 법률심은 일반법원 몫.

# 2. 담당 기관·분과(연방 헌법원 조직 표준)

- **연방 헌법원**(단일 창구)
  - **헌법재판국**: 위헌법률·헌법소원·권한쟁의·탄핵 심리·평의·선고
  - **헌법준수심의국**: 국가기관·지방·독립기관의 헌법준수 점검·권고·제도개선 의견서
  - **헌법사무국**: 접수·기록·열람·기일관리·선고공시(전자법원 연동)
  - **전원합의부(헌법)**: 중요 사건의 공개변론·전원 의견집약

> 지원 연계: **전자법원청**(제청 전자연동), **공개기록청**(익명화·공개), **사법통계국**(헌법통계·선례 태깅).

# 3. 사건 유형별 관할·요건(입구 기준)

## 3-1. 위헌법률심판
- **관할**: 연방 헌법원 전속.
- **입구**: (A)**법원 제청** 또는 (B)**직접제소**.
- **적법요건**:
  1) **직접성**: 해당 규범이 **사건 해결의 전제**일 것(제청형), 또는 규범 그 자체가 기본권을 직접 제한(직접제소형).  
  2) **현재성**: 심리 시점에 유효·적용 가능할 것.  
  3) **중대성**: 위헌 여부가 사건 결론·동종 사건군에 **결정적 영향**.
- **판시**: **위헌/합헌(O/X)** + 필요시 **잠정적 적용정지/보류** 명시.

## 3-2. 헌법소원
- **관할**: 연방 헌법원.
- **대상**: (i) 공권력의 처분·부작위로 인한 **기본권 침해**, (ii) 법률 그 자체로 인한 **직접·현재의 침해**.
- **청구인 적격**: 자기관련성·현재성·보충성(다른 구제수단이 없거나 기대 불가능) 충족.
- **기간**: 침해 통지 또는 인지일로부터 **X개월** 이내(법률로 수치화).

## 3-3. 권한쟁의심판
- **관할**: 연방 헌법원.
- **당사자**: 국가기관·지방정부·독립기관 등 공권주체.
- **쟁점**: 권한의 존부·범위·행사의 적법성.

## 3-4. 탄핵심판
- **관할**: 연방 헌법원.
- **입구**: 헌법·법률에 따른 **탄핵소추 의결** 접수.
- **판시**: **파면 여부**(O/X) + 피선거권·공직제한 기간 등 법률상 효과.

> **선거·투표·정당·공직윤리·부패 관련 위헌심사**: 선거법·정당법·윤리·부패 관련 규범에 대한 위헌심판은 **집중 심리** 대상으로 분류(단기 기한·우선 배당).

# 4. 사건 유형별 재판 진행 절차(표준 플로우)

## 4-1. 제청형 위헌법률심판(구체적 규범통제)
1) 일반법원 심리 중 위헌 의심 → 2) **제청 결정**(요건 체크) → 3) 전자제청(**전자법원청** 연동) → 4) 헌법원 접수·사전심사 → 5) 서면심리/공개변론(필요시) → 6) **선고(O/X)** → 7) 일반법원 본안 계속·최종판결(대법원까지) → 8) **위헌 후속 정비(SLA)** 자동통지

## 4-2. 직접제소형 위헌법률심판(추상/구체 혼합)
1) 청구서 제출 → 2) 적법심사(직접성·현재성·중대성) → 3) 본안 심리 → 4) **선고(O/X)** → 5) 후속 정비 통지

## 4-3. 헌법소원
1) 청구 접수(기간·보충성 심사) → 2) 본안 회부 → 3) 서면심리/공개변론 → 4) **구제명령/기각** → 5) 필요 시 입법·행정 개선 의견서 병기

## 4-4. 권한쟁의심판
1) 심판청구 접수 → 2) 당사자 지위·쟁점 특정 → 3) 서면심리·구두변론 → 4) **권한확정** 선고 → 5) 집행·통지

## 4-5. 탄핵심판
1) 탄핵소추 의결 접수 → 2) 기록 송부·변론준비 → 3) 공개변론 → 4) **파면 여부** 선고 → 5) 선고 즉시 효력 발생·통지

# 5. 헌법재판의 기본 원칙(운영 규범)
1) **집중형 원칙**: 모든 헌법재판은 연방 헌법원 단일 창구에서 담당.
2) **O/X 한정 판시**: 규범의 위헌/합헌, 권한확정, 파면 여부 중심으로 간명한 판시.
3) **헌법합치 통치**: 위헌 확정 시 사건 적용 배제 + 입법·행정·황실의 **정비 의무**(기한법으로 담보).
4) **공개·익명화·데이터화**: 선고문 공개, 익명화, 메타태깅(유형/대상/제청/기한) 의무.
5) **신속성(SLA)**: 제청·접수→선고까지 단계별 기한. 선거·정당·윤리·부패 관련 사건은 **우선 배당·단기 선고**.
6) **전자사법**: 전자제청·전자기록·원격변론 기본, API로 일반법원 사건페이지와 자동 연동.

# 6. 유형별 세부 진행 절차(체크리스트)

> 각 절차는 **역할(주체) → 입력서식 → 심리 방식 → 기한(SLA) → 산출물 → 후속 연계** 순.

## 6-1. 제청형 위헌법률심판(재판부용)
- **역할**: 일반법원 재판부(심급 불문) → 연방 헌법원
- **입력서식**: [제청서-01] 규범ID·조문·적용범위, [요건체크-01] 직접성/현재성/중대성 체크박스, [사실관계 요약-01]
- **심리 방식**: 서면 우선, 필요 시 공개변론(당사자·참고인 진술)
- **기한**: 제청결정 후 **전송 ≤ 3일**, 헌법원 사전심사 **≤ 14일**, 본안 선고 **≤ X일**
- **산출물**: **선고문(O/X)**, 결정요지 카드, 반대·보충의견(있을 시)
- **후속**: 일반법원 본안 재개, 사건 페이지에 O/X 메타 반영 → 판결 → (위헌 시) 입법·행정 정비 통지

## 6-2. 직접제소형 위헌법률심판(청구인/대리인용)
- **역할**: 청구인 → 헌법사무국 접수 → 헌법재판국
- **입력서식**: [청구서-11], [침해경로도-11], [증빙목록-11]
- **심리 방식**: 적법요건 1차 심사(보충성·기간 포함) → 본안 서면심리 → 공개변론 선택
- **기한**: 접수 통지 **≤ 3일**, 적법심사 **≤ 21일**, 본안 선고 **≤ X일**
- **산출물**: 선고문(O/X), 정비 권고서(필요 시)
- **후속**: 공개기록청 게시, 관련 일반사건 자동 알림

## 6-3. 헌법소원(기본권 침해)
- **역할**: 청구인 → 헌법사무국 → 재판국
- **입력서식**: [소원서-21], [침해사실 진술-21], [보충성 진술-21]
- **심리 방식**: 기간·보충성·자기관련성 심사 → 본안 심리(서면 중심)
- **기한**: 접수 통지 **≤ 3일**, 적법심사 **≤ 30일**, 본안 선고 **≤ X일**
- **산출물**: 인용/기각 결정, 구제명령(취소·정지·재심리 명령 등)
- **후속**: 관계 기관 통지, 제도개선 의견서 병기 가능

## 6-4. 권한쟁의심판
- **역할**: 당사 기관들 ↔ 헌법원
- **입력서식**: [심판청구서-31], [권한대비표-31], [충돌행위 특정서-31]
- **심리 방식**: 문서심리 + 구두변론(기관 대리인 출석)
- **기한**: 사건성립 확인 **≤ 14일**, 본안 선고 **≤ X일**
- **산출물**: **권한확정** 결정(존부·범위), 집행지침(필요 시)
- **후속**: 총의회·행정부·해당 기관에 즉시 통지

## 6-5. 탄핵심판
- **역할**: 소추기관 → 헌법원
- **입력서식**: [소추의결서-41], [소추사유서-41], [증거목록-41]
- **심리 방식**: 공개변론 원칙, 증거조사·당사자신문
- **기한**: 기록 정리 **≤ 14일**, 변론기일 지정 **≤ 30일**, 선고 **≤ X일**
- **산출물**: **파면 여부(O/X)**, 부수효과(피선거권 제한 기간 등)
- **후속**: 즉시 효력 발생, 선거관리·인사·윤리기관 통보

## 6-6. 선거·정당·윤리·부패 연계 사건(우선 배당 패키지)
- **우선도**: 최상
- **연계 규정**: 일반사건은 **국가·전국 트랙(고등→중앙→대법)**, 헌법심판은 **우선 심리·단기 선고**
- **공개**: 선고요지 카드 즉시 공시, 질의응답(FAQ) 동시 게시

## 6-7. 선고 후 공표·동기화(공개기록청·해석국·양형위)
- **체크**: [ ] 익명화 적용 [ ] 메타태그(유형/대상/제청/기한) [ ] 판례번호 부여  
- **동기화**: 헌법원 선고 → (**해석국**) 유권해석 보정 → (**양형위**) 기준 영향 여부 표기 → (**중앙/대법**) 동일쟁점 알림

# 부록 B. 입법·위임 체계(헌법/연방법률/황제령)

## B-1. **연방 헌법**(수록 위치 권고)
- **제○조(헌법재판의 집중)** — 제6장 제2절 헌법재판.
- **제○조(헌법 제청)** — 제6장 제2절, 제청 3요건과 O/X 한정 판시.
- **제○조(위헌확정의 효력과 후속의무)** — 제6장 제2절 말미, 제4·5장과 교차.

## B-2. **연방 법률**(제정/개정)
- **헌법재판 절차 법** — 유형·요건·기한(SLA)·공개·익명화.
- **전자법원 운영 법** — 전자제청·API·연동.
- **판례공표·해석 연계 법** — 공개기록·선도판결·해석국/양형위 동기화.
- **선거·정당·윤리·부패 특례 법** — 우선 배당·단기 선고·증거 특례.

## B-3. **연방 황제령**(시행령·규칙)
- **제청 표준서식·메타데이터 고시** / **선고기한 지침(SLA)** / **공개·익명화 세부기준** / **우선배당 리스트(선거·정당·윤리·부패)**.
